# 🔐 Form Validator

A simple and beginner-friendly Form Validator built using **HTML, CSS, and JavaScript**. This project validates email and password fields using **Regular Expressions (Regex)** and displays appropriate error messages when the input is invalid.

## ✨ Features

* Email validation using Regex
* Password validation using Regex
* Error messages for invalid inputs
* Success message when all fields are valid
* Modern and responsive UI
* Beginner-friendly code structure

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Regular Expressions (Regex)

## 📋 Validation Rules

### Email

* Must contain `@`
* Must contain a valid domain
* Example: `user@example.com`

### Password

* Minimum 8 characters
* At least one uppercase letter
* At least one lowercase letter
* At least one number
* At least one special character

Example:

```text
Password@123
```
